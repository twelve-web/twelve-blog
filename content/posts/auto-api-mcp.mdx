## 背景

因每次前端开发时每次复制粘贴 Yapi 的后端接口和手动创建 TS 类型和 Mock 数据比较麻烦因此开发自动化 MCP，使用 Typescript SDK 开发。

## github 地址 请不要忘记 star 哦～

[https://github.com/twelve-web/auto-api-mcp](https://github.com/twelve-web/auto-api-mcp)

## 安装方式

```json
{
  "mcpServers": {
    "auto-api-mcp": {
      "command": "npx",
      "env": {
        "YAPI_COOKIE": "_yapi_token=your_token; _yapi_uid=your_uid",
        "BASE_URL": "xxxxx"
      }
    }
  }
}
```

## 使用方式 一句话即可

@https://fed.qschou.com/project/810/interface/api/cat_2813 帮我生成 这是例子@index.js 加上 interface 和 mock 数据并应用
